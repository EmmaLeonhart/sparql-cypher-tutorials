## Project Description
A hands-on tutorial series for SPARQL and Cypher graph query languages. Each lesson has explanations, executable queries, and exercises using real Wikidata and Neo4j data.

## Architecture and Conventions
- **Structure:**
  - `sparql/01-basics.md` through `sparql/08-real-world.md` -- SPARQL track
  - `cypher/01-basics.md` through `cypher/06-real-world.md` -- Cypher track
  - `exercises/` -- Solutions to exercises
  - `diagrams/` -- Visual explanations of graph concepts
- **Convention:** Each tutorial file is self-contained and runnable
- **SPARQL endpoint:** Wikidata Query Service (https://query.wikidata.org/sparql)
- **Cypher:** Neo4j Sandbox examples
